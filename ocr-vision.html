Notes & caveats

Detection: uses TF.js COCO-SSD (lite_mobilenet_v2). The COCO book class is trained mostly on books seen face-on, but generally fires on spine-out books too. If detection misse[...]

Rotation: if the cropped region is taller than wide (typical for spine-out books with vertical text), it's auto-rotated 90° clockwise before OCR. If results look reversed, try re-captu[...]

Tesseract is fast, runs well offline after a one-time language data download, and produces per-word confidence. It can struggle with stylized type and tight letter spacing.

TrOCR is a transformer model trained for printed text lines — usually better on stylized typography but slower and ~25× the download. The small variant is single-line; multi-li[...]

Both run client-side; nothing leaves the device. First run downloads model weights (browser cache makes subsequent loads fast).
